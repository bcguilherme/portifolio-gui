# Guilherme Barros Correia - Model Portfolio

Professional, minimalist, and agency-ready portfolio website.

## Features
- **Agency-Ready Hero**: Face-dominant crop, bold typography.
- **Curated Portfolio**: Editorial and commercial work showcase.
- **Responsive**: Mobile-first design.

## Tech
- HTML5
- CSS3 (Variables, Grid, Flexbox)
- Vanilla JS

## Deployment
This project is configured for automatic deployment via Netlify.
Changes pushed to `main` branch are automatically built.
